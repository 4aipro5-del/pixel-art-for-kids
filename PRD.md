# PRD

## 개요

마우스로 한 칸 혹은 여러 칸 색을 채워서 그림을 그리는 픽셀아트 웹페이지 생성

---

## 목표 및 대상

초등학생들이 로그인 없이 자기 이름 입력 후 접속함

---

## 핵심 기능

- 픽셀 칸을 클릭하면 원하는 색을 채워 그림을 그릴 수 있도록 함. 
- 색칠한 칸을 다시 누르면 지우개 기능이 활성화됨.
- 완성한 이미지(png) 저장 기능, 내가 저장한 그림들을 볼 수 있는 스케치북 기능, 담벼락에 내가 그린 그림을 업로드할 수 있는 기능 추가

---

## 화면 구성

1페이지 - '픽셀아트'라고 적힌 제목과 내 이름을 적고 들어가는 맨 처음 페이지
2페이지 - 화면 크기(정사각형, 9:16, a4, 자유 지정), 픽셀 개수를 고를 수 있음
3페이지 - 브러시(1칸, 4칸), 색팔레트, 색깔 세분화팔레트, 색을 따올 수 있는 4가지 기능을 넣은 버튼이 상단에 있음. 뒤로가기, 앞으로가기 버튼 있음.
